### Was ist interne Verlinkung in HTML?

Interne Verlinkung in HTML bezeichnet das Setzen von Hyperlinks zwischen verschiedenen Seiten einer Website.  
Diese Links helfen Besuchern, relevante Inhalte leichter zu finden, und ermöglichen es Suchmaschinen, die Seiten besser zu indexieren.  
Typische HTML-Links sehen folgendermaßen aus:

```html
<a href="/kontakt">Kontakt</a>
```

Hier wird ein interner Link erstellt, der den Besucher zur Kontaktseite der Website führt.  
Für SEO ist es wichtig, dass diese internen Links gezielt auf relevante Keywords gesetzt werden, damit Suchmaschinen die Relevanz der verlinkten Inhalte erkennen können.

### Wie erstelle ich interne Links?

Die Erstellung von internen Links ist traditionell ein manueller Prozess.  
Jedoch erleichtert mein Tool die Arbeit, indem es automatisch relevante Keywords im Text erkennt und diese mit passenden internen URLs verlinkt.  
Der gesamte Prozess gliedert sich in drei Hauptschritte: das Crawlen der Website, die Keyword-Extraktion und die automatische Linksetzung.

#### 1. Crawling-Prozess

Zunächst wird die Website gecrawlt, um alle relevanten URLs zu extrahieren.  
Dies geschieht mithilfe von Python-Bibliotheken wie `BeautifulSoup` und `Requests`.  
Das Tool sammelt alle internen Links einer Website, um später die relevanten Seiten für die Verlinkung zu identifizieren.

#### 2. Keyword-Extraktion

Nachdem die URLs gesammelt wurden, analysiert das Tool den Inhalt jeder URL und extrahiert Keywords.  
Diese Keywords werden anhand von OpenAI-Technologien identifiziert, um die relevantesten Begriffe für die interne Verlinkung auszuwählen.  
Dabei unterscheidet das Tool zwischen Short-Tail- und Long-Tail-Keywords, um eine ausgewogene Verlinkung zu gewährleisten.

#### 3. Automatische Linksetzung

Im letzten Schritt wird der Text auf relevante Keywords geprüft.  
Sobald ein passendes Keyword gefunden wird, verlinkt das Tool dieses automatisch mit der entsprechenden internen URL.  
Dabei achtet es darauf, dass die Links sinnvoll im Text platziert werden und keine doppelten Verlinkungen entstehen.  
Das Ergebnis ist ein optimierter Text mit automatisch gesetzten internen Links, der sowohl für Benutzer als auch für Suchmaschinen leicht zugänglich ist.

```python
def generate_hyperlinked_text(input_text, keyword_url_pairs):
linked_keywords = set()
url_link_count = {}
# Sort keyword pairs by length to prioritize longer matches
keyword_url_pairs.sort(key=lambda x: len(x['Keyword']), reverse=True)
def replace_keyword(match):
keyword = match.group(0)
for pair in keyword_url_pairs:
url = pair['URL']
if keyword.lower() == pair['Keyword'].lower() and url_link_count.get(url, 0) < 2:
linked_keywords.add(keyword.lower())
url_link_count[url] = url_link_count.get(url, 0) + 1
return f'<a href="{url}">{keyword}</a>'
return keyword
processed_text = re.sub(r'\b(?:' + '|'.join(re.escape(pair['Keyword']) for pair in keyword_url_pairs) + r')\b', replace_keyword, input_text, flags=re.IGNORECASE)
return processed_text
```

Dieses Python-Snippet zeigt, wie das Tool die relevanten Keywords erkennt und automatisch die Links setzt.  
Es wird darauf geachtet, dass jedes Keyword nur einmal im Text verlinkt wird und keine doppelten Links entstehen.  
Das Ergebnis ist eine perfekt optimierte interne Verlinkung, die den SEO-Faktor erheblich verbessert und die Benutzerfreundlichkeit Ihrer Website steigert.
