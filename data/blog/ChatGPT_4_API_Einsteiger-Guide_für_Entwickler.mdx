## Parameter und Beispielcode

In der Nutzung der ChatGPT API sind Parameter von entscheidender Bedeutung, um die Interaktion und die Antworten der KI zu steuern. Zu den zentralen Parametern gehören "max_tokens" und "temperature", die das Verhalten und die Ausgabe der API beeinflussen.

"max_tokens" definiert die maximale Anzahl von Token (Wörtern und Zeichen), die in der Antwort ausgegeben werden. Dies ermöglicht Ihnen, die Länge der Antwort zu steuern und sicherzustellen, dass diese in den Kontext Ihrer Anwendung passt.

Ein Beispiel wäre die Begrenzung der Antwort auf 500 Token, um prägnante und fokussierte Informationen zu erhalten.

"Temperature" hingegen ist ein Parameter, der die Kreativität und Variabilität der Antworten steuert. Ein niedriger Temperaturwert (nahe 0) führt zu deterministischen und konsistenten Antworten, während ein höherer Wert die Antworten kreativer und unvorhersehbarer macht.

Dies ist besonders nützlich, wenn Sie kreative Anwendungen entwickeln, bei denen Vielfalt in den Antworten erwünscht ist.

Zusätzlich zu diesen grundlegenden Parametern können Sie auch die "messages"-Parameter verwenden, um verschiedene Rollen in der Konversation zu definieren. Hierbei gibt es drei Hauptrollen: 

1.

**Systemrolle**: Diese Rolle wird verwendet, um den Rahmen der Konversation festzulegen. Beispielsweise könnte eine Systemnachricht den Assistenten anweisen, sich als Tutor zu verhalten.
2.

**Benutzerrolle**: Diese Rolle repräsentiert Anfragen oder Kommentare der Benutzer und ist entscheidend, um den Kontext der Frage zu vermitteln.
3. **Assistentenrolle**: Die Antworten des Modells übernehmen automatisch die Rolle des Assistenten und enthalten die vom Modell generierte Antwort auf die Benutzeranfrage.

Ein detailliertes Codebeispiel zur Verwendung der ChatGPT API könnte wie folgt aussehen:

```python
import openai

openai.api_key = 'YOUR_API_KEY'

response = openai.ChatCompletion.create(
  model='gpt-3.5-turbo',
  messages=[
    {'role': 'system', 'content': 'Du bist ein hilfreicher Assistent.'},
    {'role': 'user', 'content': 'Was kannst du mir über die ChatGPT API erzählen?'}
  ],
  max_tokens=500,
  temperature=0.7
)

print(response.choices[0].message['content'])
```

In diesem Beispiel wird der Assistent so konfiguriert, dass er als hilfreicher Berater agiert, und die Antwort wird auf maximal 500 Token begrenzt.
